Ordered list:
  1. ol: oredered list.
  2. ol is a list with numbers.
  3. ol must have a closing tag but the li doesn't have to.
UnOrdered list:
  • ul: unordered list.
  • ul is a list with dots.
  • ul must have closing tag but li doesn't have to.
  • also the p closing tag is optional like the list.
  • if you leave an ordered/unordered list without a closing tag it will never know when to stop.
